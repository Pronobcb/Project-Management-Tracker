# Project-Management-Tracker
📌 Project Overview
The Task Management Tracker is an interactive Excel dashboard designed to give teams and managers a clear, real-time view of all ongoing, completed, and overdue tasks. It provides visual summaries by owner, team, and status — making it easy to identify bottlenecks and ensure accountability.

## 🚀 Key Features

- Status Tracking — Visual color-coded status badges (Completed, In Progress, Not Started)
- Overdue Alerts — Instantly flags tasks that have passed their due date
- Task by Owner — Bar chart showing task distribution across team members
- Tasks by Team — Department-wise task breakdown (Finance, HR, Strategy, Marketing)
- KPI Summary Cards — At-a-glance metrics for quick decision-making
- Detailed Task Table — Full list with start date, due date, duration, and comments

🎯 Business Value
This tracker helps teams and managers:

- ✅ Get instant visibility into task progress across departments
- ✅ Identify overdue tasks and take corrective action quickly
- ✅ Balance workload distribution among team members
- ✅ Improve accountability with clear task ownership
- ✅ Make informed decisions with summary KPI metrics
